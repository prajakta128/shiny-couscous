# 📖 Overview

**Health Platform** is a comprehensive healthcare and wellness web application designed to make healthcare information more accessible and organized.

The platform combines health education, AI assistance, hospital discovery, wellness resources, reminder management, and government healthcare schemes into a single digital solution.

Whether users need health guidance, nearby hospitals, medicine reminders, or information about healthcare benefits, this platform provides everything in one place.

---

# 🚨 Problem Statement

Many people struggle to:

* Access reliable healthcare information
* Find nearby hospitals and medical facilities
* Remember medications and appointments
* Discover government healthcare benefits
* Get basic health guidance quickly
* Maintain mental wellness and healthy habits

Healthcare information is often scattered across multiple websites, making it difficult to access when needed.

---

# 💡 Our Solution

The Health Platform provides a centralized healthcare ecosystem that allows users to:

* Learn about diseases and wellness topics
* Find hospitals and healthcare services
* Set medication reminders
* Explore government healthcare schemes
* Access emergency healthcare information
* Interact with an AI health assistant

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/prajakta128/shiny-couscous.git

cd shiny-couscous
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

---

## Open Browser

```text
http://localhost:8080
```
