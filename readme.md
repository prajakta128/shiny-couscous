# Health Platform

A unified health and wellness web app with AI chatbot, educational content, government schemes, hospital info, reminders, and mental wellness.

## Features

- AI Health Chatbot
- Disease & Health Topic Guides
- Government Health Schemes
- Hospital Scheme Finder
- Medicine & Appointment Reminders
- Mental Wellness Articles

## Run

```bash
python app.py
Visit: http://localhost:8080
```
